## What is this 3D scene ?

This is a 3D scene made with openGL. 

## what's in this 3D scene ?

In this 3D scene there is:
- 3D model loading
- Cubemaps
- Back face culling
- Frustum Culling
- Blinn-Phong shading model
- Framebuffer
- Bloom
- instancing
- Normal map
- Shadow Map
- Deferred rendering
- SSAO
- Gamma correction
